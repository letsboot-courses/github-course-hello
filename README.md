# Hello World

Einfache Hello World Applikation mit SpringBoot.

## Build

```
mvn clean install
```

## Run

```
mvn spring-boot:run
```

Optionally you can set the APP_GREETER environment variable.

## Check

```
curl http://localhost:8080/hello
```

Cypress
```
npm install cypress --save-dev
npx cypress run
```
